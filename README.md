# 介绍
dynamic-datasource-spring-boot基于spring boot采用注解的方式自动切换数据源，并且集成druid监控配置

# 安装
- git clone https://github.com/tswine/dynamic-datasource-spring-boot.git
- cd dynamic-datasource-spring-boot && mvn install
- hava fun
